# dot-files

```
brew install bat

pip install neovim && \
  pip3 install neovim
```

## Setting up coc

```
python3 -m pip install --user --upgrade pynvim
ln -s ~/dot-files/coc-settings.json .config/nvim/
gem install solargraph
solargraph bundle
```

## Fonts
1. Double click "Hack Regular Nerd Font Complete.ttf" to install
2. Set that font in iTerm
